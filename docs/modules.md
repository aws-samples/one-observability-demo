# API Documentation

The full API documentation is auto-generated by [TypeDoc](https://typedoc.org/) and published to GitHub Pages.

To build locally:

```bash
npm ci
cd src/cdk && npm ci && cd ../..
npx typedoc
```

Output is written to `./wiki-docs/`.
